在实例中学习——从零开始学JAVA #4 循环和穷举

while 循环

会一点英语的人都知道while是什么意思——‘当’。
没错这种循环就叫当循环。
基本结构：while(条件) {
//进行处理
//处理过程将会不断重复，只要条件成立。
}

do … while循环

基本结构：do {
//进行处理
//处理过程将会不断重复，只要条件成立。
} while(条件); 注意这里有个分号！

do … while循环和while循环的不同之处在于，无论条件是否成立，do … while循环中的语句都会被至少执行一次。

这里需要处理的过程比较简单，所以条件和while循环是一样的；但是在这里条件是执行完代码块再判断的，在之后的开发中需要推演一下防止条件出错。

for循环

基本结构：for (初始化;条件;每步对变量的操作) {
//处理过程
}
for循环的”初始化”部分可以定义新的局部变量，这个变量只会在for循环内生效。
for循环可以把一些和处理主过程无关、而是循环所带来的语句放在一起，因此也比较常用。缺点是，如果条件比较复杂，写在一起会引起阅读上的一定困难，这时候常常使用while循环

接下来讲讲穷举

穷举的基本结构是循环套分支，利用计算机算的比较快这一特点，循环所有可能情况，判断是否满足条件并进行相应处理。

第二课练习答案：
所有的标识符：Main main args System out println
所有的修饰符：public(两个) static

在实例中学习——从零开始学JAVA #3 基本数据类型和分支结构

Java中的基本数据类型包括：boolean char byte short int long float double
这些基本类型都是值类型。何为值类型？就是你将它作为函数参数，函数不能改变它的值。

  • boolean只能为true或者false，可以直接用于判断条件中

  • char 代表一个字符，可以用如’a’来表示一个char类型的常量。Java采用Unicode编码，因此char占用两个字节，也允许直接使用中文而不必如C++那样进行额外的操作。

    • 常见的特殊字符：’\r’’\n’代表换行/回车。值得注意的是，Windows中的换行需要一个’\r’+一个’\n’，而Linux中(包括Android)只需要’\n’
  • byte short int long 代表整数，可正可负。具体范围不用记，只需要知道大概；一般情况下都用int就行了。
    • byte从-128到127，占用1字节空间
    • short从-32768到32767，占用2字节空间
    • int从-2147483648,到2147483647共10位，占用4字节空间
    • long从-9223372036854775808到9223372036854775807共19位，占用8字节空间
  • float double代表浮点数，用来储存和计算小数，或是精确度要求不高情况下大数的运算
    • float 范围大约是10^38，占用4字节空间，精确度大概能到6位小数
    • double 范围大约是10^308，占用8字节空间，精确度大概能到12位小数
    • float的精度、范围都比double要小，但是因为6位小数的精度和10^38的范围在游戏中一般是够用的，而储存空间又占用较小，因此游戏引擎中多数使用的是float类型。
    • Java默认如1.23是一个double类型，如果你写float a = 1.23;会编译报错。你必须加一个f或者F作为后缀，表示这是个float类型的小数常量。
    • 小数常量有多种类型。1.23是通常的表示方法，当整数部分为0是也可以直接.23省略0；可以用科学计数法表示，比如1E10=10^10。

分支结构是一个程序中不可缺少的部分，结构本身有一个入口，在途中分为两条或多条”岔路”，随条件不同而进入不同的”岔路”。随后在某处岔路会合并【哪怕是在结束时】，所以分支结构只有一个出口。
怎么感觉讲得太概念了，直接上例子吧。

如果你有兴趣尝试的话，你可以试一下10.1；准确值是102.01，但是输出的是102.01001，这就是浮点数有精度限制的体现。

另一种分支：switch … case …

备注：Java7中加入了switch的值是字符串的时候的情形，但是有诸多注意事项，在这里不作阐述，不推荐使用。

在实例中学习——从零开始学JAVA #2 基本语法

这一课将会分几个部分来阐述。只讲一些基本概念，方便之后阐述问题。

1、类(class)

上一课中，我们讲了一个A+B Problem的例子，其中创建了一个类，Main。
类是Java开发的一个基本元素，可以用来储存自定义数据结构、在其中编辑代码逻辑等。

2、修饰符(modifier)

继续看上一课中的例子。

其中修饰main函数和Main class的public、static就是修饰符。除此之外修饰符还有private protected final abstract等

3、标识符(token)

Java所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符。例子中我们定义的标识符有：
Main(类名) main(方法名) args(变量-函数参数) scanner a b(变量-局部变量)
注意：只有类名与文件名完全相同时，这个类才能修饰为public。
Java是大小写敏感的，这意味着标识符a和A是不同的。

4、注释

注释是为了让别人（或者自己）能读懂代码而附加的，或是用来附带一些其他信息如作者等，在编译时会被忽略。
//开头的是单行注释，如上例中”输出 a + b的值”
用/ /包含的是多行注释【当然也可以在一行中使用】

在实例中学习——从零开始学Java #1 输入输出

首先，你需要下载并安装开发环境。

  1. JDK，去官网上下即可，http://www.oracle.com/technetwork/java/javase/downloads/index.html
  2. 这里用的是Eclipse，各种地方的都可以。不建议从官网Installer下，因为慢。

打开Eclipse，会出现一个欢迎界面，如下图。不用管它，Window-New Window即可。

在左侧的Project Explorer中空白位置右键->New->Java Project，输入新工程的名字，单击Finish

按工程名前面的+号，展开新的工程，右键src->New->Class，名字取为Main，并勾选public static void main选项（这是程序入口，其他类不需勾选），单击Finish

（第一次讲详细些，之后就略去细节了！）

关键函数：System.out.println System.out.print
关键类：Scanner

PS：一个很好用的功能，自动补全，可以在设置中开启。菜单栏Window->Preferences->Java->Content Assist，下面有个Auto activation triggers for Java，把a~z，A~Z还有数字0~9都加进去。
